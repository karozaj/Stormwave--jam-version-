STORMWAVE

They fought valiantly. They all fell to the demons. You're the last one standing.
The attacks seem to have let up, but make no mistake - this is just the calm before the storm.
Prepare. Fight. Survive.

CONTROLS:

Mouse - look around
WASD - move
Shift (hold) - sprint
Space - jump
LMB - shoot/destroy blocks
RMB - place blocks
1-5/Scroll wheel - select weapon
E - open ammo shop (build mode only)
Esc/P - pause menu

This game was made for Brackeys Game Jam 2024.2 on itch.io
